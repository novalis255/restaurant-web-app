# Sistem Informasi Restoran

Aplikasi sistem informasi restoran berbasis web menggunakan:

- PHP Native
- MySQL
- Docker Desktop
- phpMyAdmin
- CSS manual

## Role
- Admin
- Kasir
- Pelayan

## Cara Menjalankan
1. Install Docker Desktop
2. Clone repository
3. Jalankan `docker-compose up -d`
4. Akses `http://localhost:8000`

## Login Default
- Admin: admin / admin123
- Kasir: kasir / kasir123
- Pelayan: pelayan / pelayan123
